# scala-web-maven-spring-swagger-hello-world

## Description
A POC for swagger in springframework.

## Tech stack
- scala
- maven

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- API http://localhost/hello
- Raw JSON http://localhost/v2/api-docs
- Swagger http://localhost/swagger-ui.html

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
